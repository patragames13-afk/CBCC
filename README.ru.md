# CBC Ballistics

Баллистический калькулятор для мода **Create Big Cannons** в Minecraft.
Рассчитывает углы возвышения/поворота, траекторию, время полёта,
разброс и дульную скорость для пушечных снарядов.

## Возможности

- **Двойная траектория** — показывает решение прямой наводкой и мортирой
- **Сравнение боеприпасов** — pitch/flight/apex для всех типов аммо
- **360° компас** — азимут в градусах по часовой от севера
- **Тёмный GUI** — интерфейс на GTK3 с тёмной темой и графиком
- **Пресеты** — сохранение/загрузка конфигураций в файл
- **Копировать** — экспорт отчёта одной кнопкой
- **CLI режим** — интерактивная консоль с `--cli`

## Сборка

Требуется: `cmake`, `g++`, `libgtk-3-dev`

```bash
cd cbc_ballistics
cmake -B build
cmake --build build
```

## Запуск

```bash
./build/cbc_ballistics       # графический режим
./build/cbc_ballistics --cli # консольный режим
```

## Управление (GUI)

| Поле | Описание |
|-------|-------------|
| Cannon X/Y/Z | Позиция пушки в мире |
| Target X/Y/Z | Позиция цели |
| Ammo / Barrel / Dim | Тип снаряда, материал ствола, измерение |
| Barrel len | Количество сегментов ствола |
| Charges | Заряды пороха |
| Calculate | Запуск расчёта |
| Copy | Копировать отчёт в буфер |
| Save / Load | Сохранить/загрузить пресет |
